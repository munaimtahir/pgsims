# Visual Improvement Plan

## Level 1 - Quick Polish

- Increase table row padding and default font size on all admin tables
- Add stronger page subtitles that explain the purpose in plain language
- Normalize button labels so actions read like tasks, not internal operations
- Add clearer empty states with one next action
- Tone down secondary gray text on data-heavy pages
- Add more whitespace between dashboard sections

## Level 2 - Usability Redesign

- Split the current UTRMC overview into a simple overview and a dedicated onboarding/import workspace
- Make role-specific dashboards show only the work that matters for that role
- Replace the matrix checkbox grid with a friendlier editing flow
- Add preview-before-save for bulk/onboarding actions
- Add a “needs attention” panel to each dashboard
- Move hidden operational pages into the main task flow or make them discoverable from the dashboard

## Level 3 - Accessibility and Senior-Friendly Mode

- Add a larger-text mode for older and busy users
- Add a high-contrast mode for low-vision users
- Reduce table density by default
- Replace icon-only or tiny action targets with labeled controls
- Prefer “show details” expansion over dense always-open tables
- Add printable summaries for supervisors and administrators

## Priority Guidance

- Start with the resident dashboard crash and the UTRMC overview simplification
- Then fix the matrix and the admin tables that dominate the current baseline
- Finally improve accessibility and senior-user comfort
