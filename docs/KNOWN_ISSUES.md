# PGSIMS Known Issues

The following are documented non-blocking limitations, bugs, or system constraints in the PGSIMS Pilot Baseline v1.0.

## 1. Playwright E2E Timing and Token Injection
- **Symptom**: Dashboard smoke tests fail occasionally with auth/render timing errors.
- **Root Cause**: Next.js hydration timing creates a race condition where API calls execute before the E2E auth script finishes injecting simplejwt tokens to `localStorage`.
- **Workaround**: Playwright E2E helpers include a `page.waitForTimeout(100)` wait shim to permit token hydration.

## 2. SQLite Test DB Schema Locks
- **Symptom**: Local unit tests sometimes fail to initialize database.
- **Root Cause**: Django SQLite database engine locks when performing concurrent schema drop-tables on models with legacy FK relations.
- **Workaround**: Solved by directly deleting models from migrations rather than through model field modifications.

## 3. Bulk CSV UTF-8 Encoding Mismatch
- **Symptom**: Upload of CSV files generated by old versions of Excel fails with Unicode decoding error.
- **Root Cause**: Excel exports default to windows-1252 or UTF-8 BOM, but the server expects standard UTF-8.
- **Workaround**: Save files explicitly as "CSV UTF-8 (Comma delimited) (*.csv)" in modern spreadsheet tools.
