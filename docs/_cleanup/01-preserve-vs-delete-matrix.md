# Preserve vs Delete Matrix (Final for Purge Execution)

## Decision Matrix

| Area | Action | Scope | Reason |
|---|---|---|---|
| Django migrations and schema history | **PRESERVE** | `backend/**/migrations/*` | Required to rebuild trusted baseline and maintain auditability |
| Contract docs | **PRESERVE** | `docs/contracts/**` | Contract-first integration authority |
| Core governance/docs | **PRESERVE** | `AGENTS.md`, canonical operational docs | Prevent policy drift |
| Environment/config + CI/CD | **PRESERVE** | `.env*` (uncommitted), compose, workflow configs | Required runtime/bootstrap integrity |
| Canonical reference/master data model definitions | **PRESERVE** | Department/Hospital model code and related policies | Required canonical data integrity |
| Role/permission structure | **PRESERVE** | RBAC code and mappings | Required authorization baseline |
| Required bootstrap/admin accounts | **PRESERVE (RECREATE MINIMAL)** | Keep only minimum verification/pilot users | Avoid demo/test identity clutter |
| Test/demo transactional DB records | **DELETE** | Submissions, workflow rows, temporary approvals/reviews, seeded demo/e2e traces | Reset truth and remove fake/dirty state |
| Fake/test/demo accounts | **DELETE** | `demo_*`, `e2e_*`, `pg_p6-*`, `sup_p6-*`, `res_e2e-*`, ad-hoc test accounts | Remove non-pilot identity clutter |
| Stale uploads/media tied to deleted runtime state | **DELETE** | Media volume + local test uploads | Remove orphan/stale runtime artifacts |
| Generated runtime outputs | **DELETE** | `output/**`, `OUT/**`, `frontend/e2e/output/**`, stale screenshots | Remove stale evidence/debug clutter |
| Local caches/build artifacts | **DELETE** | `.next`, `.pytest_cache`, `__pycache__`, temporary logs | Ensure clean baseline rebuild |
| Redundant temporary presentation/discovery artifacts | **DELETE/ARCHIVE** | `docs/_presentation_assets/**`, `docs/_presentation_discovery/**`, presentation scripts | Non-authoritative stale packs |
| Non-authoritative stale audit bundles | **DELETE/ARCHIVE** | stale autogenerated evidence packs only | Keep active truth docs only |
| Sample seed source files | **PRESERVE** | canonical seed scripts and input CSV/XLSX | Needed for controlled reseed |
| Audit logs/history tables | **PRESERVE CONDITIONALLY** | Keep schema + ability; runtime rows may reset after backup | Audit integrity preserved by schema and snapshot |
| Notification logs | **PRESERVE CONDITIONALLY** | Preserve schema/contracts; runtime rows can reset post-snapshot | Avoid polluted demo signals |
| Evidence folders | **PRESERVE CONDITIONALLY** | Preserve only current authoritative docs; archive or delete stale packs | Avoid misleading status claims |

## Mandatory Keep Set (Must Remain)
1. Migrations and data model code.
2. `docs/contracts/**` plus governance docs.
3. RBAC and auth enforcement logic.
4. Environment/compose/build configs.
5. Canonical seed/reset commands and scripts needed for minimal rebuild.

## Mandatory Delete Set (This Sprint)
1. Runtime demo/test DB state and fake identities.
2. Stale generated media/output/screenshots/report artifacts.
3. Local runtime caches/build leftovers and temporary clutter.
4. Non-authoritative stale docs/evidence bundles that overstate capabilities.

## Conditional Preserve Decisions Applied
- **Audit/notification/runtime evidence rows**: backed up in snapshot first, then allowed to reset.
- **Demo artifacts**: not kept as active truth surface; retained only in pre-purge snapshot if needed for forensics.
