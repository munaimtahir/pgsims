# UTRMC Overview Redesign

## Problem

The old overview read like an import/setup console instead of an operational dashboard for non-technical staff.

## Fix

- Retitled the page to `UTRMC Dashboard`.
- Reframed the top section as a simple operational summary.
- Added:
  - Today’s attention
  - Setup completeness
  - Recent activity
  - Data quality status
  - Quick actions
- Moved onboarding/import entry points into a compact card and a dedicated onboarding route.

## Dedicated Route

- `/dashboard/utrmc/onboarding`
